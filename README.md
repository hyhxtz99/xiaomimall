# 小米商城官网仿站项目

## 项目概述
本项目是一个仿制小米商城官网的前端项目，旨在通过模仿小米商城的界面和功能，提升前端开发技能。项目使用了HTML、CSS技术，实现了小米商城官网的主要页面和功能。

## 项目特点
- ​**交互效果**：使用css实现了一些常见的交互效果，如下拉菜单。
- ​**页面结构**：项目按照小米商城的页面结构进行布局，包括首页、商品详情页、购物车页等。

## 技术栈
- ​**HTML5**：用于构建页面的基本结构。
- ​**CSS3**：用于页面的样式设计，包括布局、颜色、字体等。
